# Adaptive-inverse-PINN-to-determine-the-source-of-PM10-in-the-Aburr-Valley
Se tratará de conformar una PINN Inversa Adaptativa para hallar la función de fuentes emisoras de la Ecuación de Advección-Difusión-Reacción a partir de datos de concentración de PM10 dados por las estaciones del SIATA en el Valle de Aburrá
